## Add Repository to Home Assistant

1. Go to **Settings → Apps → Install App**.
2. Click ⋮ → **Repositories**.
3. Paste `https://github.com/JosephBlock/HA-Apps` and click **Add**.

## Available Apps

|-----|--------|-------------|
| RIPE Atlas Probe | [ripe-atlas](ripe-atlas/README.md) | Run a RIPE Atlas software probe from Home Assistant and print the probe public key on boot for registration. |
